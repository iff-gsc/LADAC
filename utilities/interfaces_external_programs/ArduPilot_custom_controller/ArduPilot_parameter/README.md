# Tunable MATLAB/Simulink Controller in ArduPilot (beta)
This toolchain extends [ArduPilot custom controller](../) and allows tunable controller implementations in ArduPilot. **Tunable** means, that the parameters used to parametrize the Simulink controller model are integrated into the ArduPilot parameter ecosystem and can be changed through any Ground Control Software. This eliminates the need to recompile and reflash the custom ArduPilot firmware when the custom controller parameterization is changed, resulting in a huge acceleration of the flight test workflow. However, it must be said that this implementation is hacky and must be considered **early beta**, meaning that it may result in non-compilable, or worse, compilable but crashing code.
